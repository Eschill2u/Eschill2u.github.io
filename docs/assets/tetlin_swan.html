Data Clean-Up: Historic Data

Let’s pare down the Alaska Swan Database data set to fit our needs.

First, let’s load our packages, import the data, and put it into a data frame for manipulation. Uncomment the first line to install missing packages.

map1_quads <- c("A2", "A3", "A4", "B4", "B5", "B6")

old_df <- old_df[which((old_df$Month != 5) & (old_df$Month != 6)), ]
old_df <- old_df[which(old_df$Year >= 1985), ]

…and remove unnecessary columns, such as those for nesting data (which we are not looking at) or quarter quad numbers (which were never utilized at Tetlin).

old_df <- subset(old_df, select = -c(X, QtrQuad, Nest, Map, Quad))

Data Clean-Up: New Data

Now, let’s format the new data to match the historic data.

Let’s begin again by importing.

new_df <- read.csv("newswans_2022.csv")

Next, we’ll remove null-valued rows…

new_df <- new_df[which(new_df$TOTAL != 0),]

…and remove extra or unnecessary columns.

new_df <- subset(new_df, select = -c(TOTAL, Pairs, ele, name, ID))

We also need to separate out flocks observed with other group types into separate, individual observations (rows). We are going to adjust the latitude value of these “new” observations with an increase of 0.0001.

for(i in 1:nrow(new_df)){
  count = 0
  if((new_df$GroupSize[i] > 0) && (new_df$PAIRSX2[i] > 0)){
    count = count + 1

    addRow <- new_df[i,]
    addRow$GroupSize <- 0
    addRow$PAIRSX2 <- 0
    addRow$lat <- addRow$lat + count/10000
    new_df <- rbind(new_df, addRow)
    new_df$Singles[i] <- 0
  }
}

To match the historic data, we’ll add columns for the numbers of adults and young per observation.

new_df$Adults <- new_df$GroupSize + new_df$PAIRSX2 + new_df$Singles
new_df$Young <- new_df$Cygnets

We also need to create a new symbology column to identify the group type for each observation. The abbreviations used are as follows: “S” for single no brood, “P” for pair no brood, “SBr” for single with brood, “PBr” for pair with brood, “F” for flock, and “0Br” for lone cygnets.

new_df$Leg_Sym <- NA
for(i in 1:nrow(new_df)){
  if(new_df$Singles[i] > 0){
    if(new_df$Cygnets[i] > 0){
      new_df$Leg_Sym[i] <- "SBr"
    }else{
      new_df$Leg_Sym[i] <- "S"
    }
  }else if(new_df$PAIRSX2[i] > 0){
    if(new_df$Cygnets[i] > 0){
      new_df$Leg_Sym[i] <- "PBr"
    }else{
      new_df$Leg_Sym[i] <- "P"
    }
  }else if(new_df$GroupSize[i] > 0){
    new_df$Leg_Sym[i] <- "F"
  }else{
    new_df$Leg_Sym[i] <- "0Br"
  }
}

Now, we can remove columns that were captured by the new symbology column.

new_df <- subset(new_df, select = -c(Singles, PAIRSX2, Cygnets, GroupSize))

We will separate out the date column into individual columns for the day, month, and year.

new_df$Month <- as.numeric(substr(new_df$time, 6, 7))
new_df$Day <- as.numeric(substr(new_df$time, 9, 10))
new_df$Year <- as.numeric(substr(new_df$time, 1, 4))
new_df <- subset(new_df, select = -time)

The final tweaks require adding a contributor number column and capitalizing the “lat” and “lon” column headers for consistency.

colnames(new_df)[which(colnames(new_df) == "lat")] <- "Lat"
colnames(new_df)[which(colnames(new_df) == "lon")] <- "Lon"
new_df$Contrib <- rep(108,times=nrow(new_df))

Retrieve New Summary Calculations

summarize <- function(df){
  years <- sort(unique(df$Year), decreasing = FALSE)
  years <- years[which(years >= 1985)]
  
  total_swans <- c()
  total_adults <- c()
  breeding_adults <- c()
  total_broods <- c()
  total_cygnets <- c()
  
  for (val in years){
    total_swans <- append(total_swans, sum(df$Adults[which(df$Year == val)], df$Young[which(df$Year == val)]))
    total_adults <- append(total_adults, sum(df$Adults[which(df$Year == val)]))
    breeding_adults <- append(breeding_adults, sum(df$Adults[which(df$Year == val & (df$Leg_Sym == "PBr" | df$Leg_Sym == "SBr"))]))
    total_broods <- append(total_broods, length(which(df$Year == val & df$Young > 1)))
    total_cygnets <- append(total_cygnets, sum(df$Young[which(df$Year == val)]))
  }
  
  proportion_breeding <- breeding_adults/total_adults
  avg_brood_size <- total_cygnets/total_broods
  
  annual_growth_rate <- c(0)
  for (i in 2:length(total_swans)){
    rate <- round((total_swans[i]/total_swans[i-1])^(1/(years[i]-years[i-1]))-1, digits=4)
    annual_growth_rate <- append(annual_growth_rate, rate)
  }
  
  summary_df <- data.frame(years, total_swans, annual_growth_rate, breeding_adults, proportion_breeding, total_broods, total_cygnets, avg_brood_size)
  return(summary_df)
}

Here is what that report can look like:

In 2022, the trumpeter swan census survey was flown from 8/16 to 8/18. 1103 total swans were counted during the survey, which is a 39.3% decrease since the previous survey flown in 2015. The population growth rate since the last survey was about -6.88% annually.

Of the 916 adults, 141, or 15.39%, were deemed to be breeding. The proportion of reproducing adults decreased by about 3.58% since the previous survey. There were 187 observed cygnets and 47 total broods, corresponding to an average brood size of about 3.98 cygnets.
